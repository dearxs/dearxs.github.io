使用 macOS 的 Time Machine

2021.07.07

第一次使用了苹果系统的备份工具：时间机器。确切的说，只是使用了备份的功能，还没有恢复过。过两天使用恢复的功能。

之前尝试用过一次，但当备份剩余时间显示为十几个小时的时候，我放弃了。这委屈谁受得了。今天在网上搜索了一下加速备份的方法，还真有。

命令使用过后一定要恢复原状态，因为这个命令调整的是系统服务的优先级，影响还是挺大的。命令生效需要一定的时间，所以也不用时刻盯着备份剩余时间。反正启用命令之后，我原先显示十几个小时的备份，两三个小时就备份好了。

  • 加速命令：sudo systemctl debug.lowpri_throttle_enable=0
  • 还原命令：最后那个 0 改为 1 就可以。
