mas: macOS 命令行中的 App Store

2021.06.22

macOS 上的 App Store 中安装软件，总是遇到一只转圈圈，就是不安装，重试好几次，重启机器也没有用。最后找到了这个方法，命令行的 App Store，屡试不爽，分享出来。

下面是该工具的 github 页面，能看懂的可以直接去看： mas-cli/mas: Mac App Store command line interface

看不懂的可以往下看，很简单的几个步骤。

  1. 安装工具（brew install mas）

安装好之后，如果你英文好，可以直接 mas help 看官方的指南。

  1. 搜索软件（mas search 软件名） 这条命令就是搜索你要安装的软件，比如我搜索一个叫做 xnip 的截图软件（ps 超级好用），结果如下：
$ mas search xnip
  1059334054  截图(Jietu)-快速标注、便捷分享的截屏工具       (2.3.0)
  1485844094  iShot-优秀的截图录屏工具                (1.7.7)
  1544899154  白描截图取字                         (1.1)
  1221250572  Xnip - Screenshot & Annotation (2.2.0)
  1497428978  超级右键-iRightMouse               (1.2.0)

主义每个软件前面都有一串数字，复制以下，下一步需要使用。 3. 安装软件 （mas install 识别码）

如果你之前没有购买过此软件的话，你要使用这个命令 mas purchase 识别码。然后会跳出输入密码的窗口，输入之后就会安装了。

如果你之前购买过该软件的话，你可以直接使用 mas install 识别码。就会直接安装所需的软件了。
