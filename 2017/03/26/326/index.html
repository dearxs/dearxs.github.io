关于房子和自己

发表于 2017-03-26 更新于 2020-03-17 分类于 日记 Disqus：

很不顺的日子，工作一塌糊涂地拖，生活乱七八糟的房子，跟猫猫的关系也磕磕绊绊。工作上看不到什么前景，生活上靠自己根本买不了什么房子，这么想想，似乎已经生无可恋了。

不会

和猫猫去一起去心理咨询已经几次了，其实每次去我心里都是拒绝的，因为要真正直面自己是一件很难，很残酷的事情。上次自己哭掉了，「我需要你的在意，需要你的关怀，需要你的爱」。

这次咨询师提起这个问题，问猫猫，那他已经表达过想要你的一些关心和在意，你愿意做么？愿意朝哪个方向努力一些么？「不，不愿意，我也不会。」

「也许他更优秀一些，我会去做，但我也不知道，因为他还没那么优秀。」

全都是泡沫

房子的事情是越来越看不懂了。3 月 17 日北京新政：

重新恢复了 2014 年曾推行过的“认房又认贷”，将二套房首付比例调整到 60% 到 80%，限制任何可能的炒房行为。

限购这样的政策越来越让人奇怪，就好比大蒜的价格一直涨，庄家出来说的是，手里有一头大蒜的人，买第二头大蒜要付全价；手里没的，买大蒜才会有优惠；手里没大蒜的，要买上海的大蒜，必须在上海待过五年。这样一看，限购真的只是为了稳定大蒜的价格。所以，房子现在就是稳稳还要涨是么？

忽然就想起邓紫棋那首歌《泡沫》：「全都是泡沫，都是刹那花火……」
