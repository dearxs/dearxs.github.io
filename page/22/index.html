上海买房记（一）

发表于 2017-03-20 更新于 2020-03-17 分类于 日记 Disqus：

上周末去看了上海的房子，宝山上大区域，已经离失去很远了。坐地铁到上海大学站，3 号口出来，是锦秋花园小区。外面一排门面，大概有三四家的中介，周六那天约了两个中介，上午十一点出门，以为时间还蛮充裕，后来才知道是自己太幼稚。

因为在 房天下 软件上看到 远东君悦庭 的房子标价很低，比 链家 上面低了一两百万，也就是 链家 上四五百万的，在 房天下 上面只要两三百万，所以才约了 房天下 上面的中介。

看的区域距离上海市中心距离 13 公里左右吧。

经纬城市绿洲一期

带看的是一个看起来比较小的男生，嘴皮有点外翻，土里土气。中介门面是“品信地产”，不知道和 房天下 什么关系，他没有解释，我也没有多问。

第一个看的是 经纬城市绿洲 一期“阳光水岸”的一个一居室，小男生带着我，骑着电动车，风尘仆仆，穿过大半个 锦秋花园，到了小区旁边他们公司的办事处拿了钥匙，就走去小区了。有门禁，没卡，小男生很尴尬地跑过去试图拉住刚刚进去的人打开的门，未果。等了几分钟，房子就在右手边的第二栋楼上，八层。

电梯很破，走上去，套了鞋套，一个大爷门开着等我们。房子有点旧，右手厨房，正对洗漱，左手客厅，很小，走前方有个阳台，右手再往前是主卧，主卧旁边旁边有个小小的储物室。主卧有个飘窗，望出去，一片低矮破旧的房子。

六七十平，三百五十多万。

聚丰景都 动迁房

第二个去的是 聚丰景都，比较旧的小区，很大，前面是一排排的商品房，后面有几排动迁房，这次看得是六楼的，复式，两个大露台。

去的时候有三批人在看，三个中介，带着三批人，大概有十几个人在复式的两层房子里转来转去。房子应该是粗略装修过，据说之前是二房东在出租，房东收回来敲掉自己简单装修了一下。比起经纬一期的，这个要大很多，复式，空间大，两个大的露台。

但是骑车过来的路上，两边好多卖寿衣香烛的，你已经可以感受到这周边的几个小区有多老了吧。

五十八平米，三百五十万。

风水宝坻

小伙估计对自己业务不自信，想起了自己的老地盘，顾村。所以，接下来，滴滴打车，带我去看顾村的一个小区里面的几个房子。同样是个老小区，距离上大地铁站六公里。

第一家，14 层吧，老太太在家，老大爷在床上睡着，进屋就是个客厅，有一堆小孩的的东西，小孩地盘左边是个次卧，右边是老大爷住的主卧。看完出来，我吃了玩河南烩面，小伙去拿钥匙去了。

第二家，没人住。进门客厅，左边厨房，右边卧室，还凑活。第三家，女主人下来开的门，进门男的躺在沙发上玩 iPad。好破的小区。

然后就走了，我骑了 mobike，小伙找不到小黄车，无奈还得滴滴打车。路上小伙估计看我看得失望，跟我介绍了 中国铁建 青秀城。回去就基本看不成了。没心气了。三点多了，等等猫猫就来了。约了链家看看 远东君悦庭。

锦秋花园 十七区

等了等，喝了口水，就去链家了。和一个接待的小伙聊了两句，小伙应该做了也不久，问了几个远点的 大华朗乡 问题，囫囵过去了。

什么 远东 不满两年，税费比较高；经纬四期 也是一样的情况。经纬一二期都比较久，包括三期离地铁比较远，等等瞎聊。

问了问猫猫快到了，我就去接她了。接上她和链家的一个小姑娘以及经理去看看 锦秋花园 十七区的一个二层，房子不错，很新，旁边是小河，再过去是祁连山路，对面是在建的 滨江·公园一号，也不知道滨的那条江。

九十几平米，五百万左右。

猫猫已经不要看了，心心念念要去看 远东 ，劝她顺路看看，所以又去看了 锦秋花园 九区的一套，里面有人住，一个年级和我相仿的男生，小孩在家，女主人在家，一个像是女主人妈妈的老太太在看孩子。房子和之前那个差不多，八十几平米，但是有两个南北通透的阳台。差不多也要五百万了。

远东君悦庭

走去看猫猫心心念念要看的 远东 了。确实不错，门面低调大气。

先去看了一个四层的大两室，伍佰肆拾万。确实好，门禁牛逼，电梯干净，房型更是好。阳台，大主卧，宽敞的洗漱台，像个五星级酒店的大套房。然后又看了个 14 层的大两室，价钱比上一个贵，但是没觉得比上一个好到哪里去。这两个唯一的缺点可能是距离公路比较近，会有车流的声音。14 层的那个大窗户更是正对着上大的大操场。

然后等了一会去看了，一个一室的房型，72 平，420 万。房型仍然是很好。这是客厅，有阳台，傍晚去的，光线有点暗。

这是卧室。

阳台看出去，右手是别墅区，左手高层。

怎么样，四百二十万的价格怎么样，很低是不是？呵呵哒。洗洗睡了。接着看。

「每周跟两个不熟的人吃午饭」

发表于 2017-03-15 更新于 2020-03-17 分类于 日记 Disqus：

下午和老板聊天，聊了最近买房子的事，以及我对于和人交流的困惑。当中，他顿了一下，问我，「你喜欢跟我聊天么？」「不喜欢，诚实的说，不喜欢。」他面上不动声色，但我看出来他有点震惊。

后来聊完，他也大大方方地说，你伤到我了，哈哈。回来碰到同事每个都讲，好生尴尬。从感受上来说，我真的不喜欢；但是每次和他聊完，还是能有一些收获的。他在锻炼他的教练技巧，我也蛮受用他的人生经验。

老生常谈：怎么样才能跟任何人聊得来？

当然聊天会聊到我最喜欢讲的一个我的缺点：我就是踏不出跟别人主动交流的哪一步。至于为什么，其实最近读的那本《跟任何人都聊得来》里面说的很清楚：

  • 觉得自己差劲
  • 不想麻烦别人

感觉我应该把书里面的好多点给背下来。老板最后总结下来给的建议是这么几点（我有些忘记了），

  • 真诚地展现真实的自己
  • 真诚地对对方感兴趣
  • 我能给对方带来什么
  • 对方能给我带来什么

后来不知道怎么就给我下了一个指标，就是「 每周跟两个不熟的人吃午饭 」，在我打出这几个字的时候，我是很不愿意承认是我自己接受了这个 KPI 的事实的。踏出这一步吧。

You can only GROW when you step out of your comfort zone.

每个人都是自己宇宙的中心

发表于 2017-03-13 更新于 2020-03-17 分类于 日记 Disqus：

第二次跟猫猫去做了心理咨询。两个人无法好好交流，只能通过咨询师来交流，这样一看，好失败啊自己。我还是自诩可以游刃有余地处理自己的情绪的，但是到了猫这里，真的遇到了不曾想到过的困难。好多次心里都在嘀咕，「原来还有人会这么想」。

两个人都很自我，都是从自己的角度看事情的。换位思考这个事情说起来简单，做起来真的很难。

读《跟谁都聊得来》

在 Kindle 上看了《跟谁都聊得来》这么一本书，换到以前，看到这个书名，估计我的脸就会撇到一边。现在会觉得，也是猫总提醒我的，不能太早地对一个事情做出评价。这本书看了不少，写的真的很对症，就是教内向的人如何认清楚自己的特点，找到适合自己的与人交流的方式。

有很多点说的都让我由衷地认同，甚至感到醍醐灌顶（ 就像是之前看到「80% 的对话都是在交流情感」）。比如，

聊天是为了建立联系，是为了理解别人，并不是为了炫耀自己。

我这种人很多时候不愿意，或者说不敢和别人交流，主要原因就是觉得自己没有什么亮点可以秀给别人，所以不情愿去开口。但是看到这句话，你就知道了。我的感受完全是多余的。

过两天写一个读书笔记。

不喜欢的事情就可以不做吗

发表于 2017-03-07 更新于 2020-03-17 分类于 观点 Disqus：

有这种感觉好久了。一个人无论做什么事情，只要做的多了，肯定会很熟练的。但是回到自己工作上，我还真是觉得这一点说错了。和人打交道这个事情，我还真就熟练不起来。

今天有那么一个注意飘过脑海，我跟我的实习生的想法其实没有什么两样。之前跟实习生聊得时候，他说，「肯定喜欢做自己擅长的事情啊，不擅长的东西做出来大家还指指点点的。」嘴上讲，你还是要做一些不那么擅长的事情，练练。其实回头想想，在这一点上，我比他更加严重。

对于不擅长的事情，我会逃避，会选择不去做或者最后做。这一点其实好久以来都没有改观。最近这一点更是严重地影响了我对工作的情绪：我干不下去了。

同事不配合，各种莫名其妙的求助，各种不给力的系统，乱七八糟的人际关系……再加上生活上的各种事情和烦恼，感觉自己的 CPU 已经满负荷运转了，根本无暇顾及到这个事情做没做，做的好不好了。

应该也算是拖延症晚期了，把自己不喜欢不擅长的事情放到最后，拖到最后自己还是不想去做，只能选择逃避了，最后发现只能辞职了——最终的逃避。

所以问题来了，当初 选择职业 的时候怎么就没有去选择自己喜欢的呢？到现在，八竿子打不着的去找工作，当然简历投出去杳无音信了。

需要改变。

你为什么不喜欢与人交流

发表于 2017-03-06 更新于 2020-03-17 分类于 日记 Disqus：

不喜欢现在的工作，无法发挥自己的特长，受到了自己缺点的各种折磨。

之前看一份报告里面说「社交技能好的人在未来的社会更受青睐，工资也会更高」，如此看来，估计我就是要被淘汰的那一批了。有时候也会心血来潮地喜欢去和人打交道，但是更多的时候是觉得心很累，可能是自己的病，也可能就是自己的层次低。

今天自己来试着分析一下自己。我很喜欢和有知识的人聊天的，因为可以学到很多东西。

比如之前同事吃饭的时候分享的「目光接触」的技巧：很多时候我们都会去选择逃避别人的目光，但是其实主动的目光接触是一种力量和自信的展示，也会让人感觉到尊重。这就是一个之前我隐约知道，但是不会刻意体会，刻意练习的技巧，我很高兴可以知道。

比如之前同事会和我分享她最近对于个人发展的进展，她会去和很多人聊，希望大家给她反馈，她哪里有问题，哪里做的好，适合做什么，在你心里的形象是什么样子的等等。我也很欣赏她，透彻。

我怕和人交流。为什么呢？

  1. 因为我觉得自己没有什么可以交流分享的东西。
  2. 我害怕他们知道我自己的不堪。

那再来仔细看这两个问题。

我真的没有什么东西可以分享吗？

我内心是不愿意承认的。虽然我感兴趣的东西和她们做的事情不那么一样，但人与人交流不就是交流不同的东西么？所以我是有东西分享的，只是我害怕自己说的东西她们不感兴趣（或者让他们知道我的不堪，我们下一点谈），所以我羞于张口。

我可以分享什么东西呢？比如编程，比如公众号，翻墙，搭建播客，知识变现，买房租房，出国游玩。我自己做了什么有趣的事情可能对别人有益处呢？互联网上的风潮。摩拜和 ofo 的大战。

第一个问题解决了。

我真的不堪吗？

我觉得我哪里不堪呢？我觉得我到现在还没有买房，结婚不堪么？这是最重要的一个吧。我觉得我自己的工作做不好不堪么？是的。这也是我的不堪。我觉得我没好多同学混的好不堪，也是这样的。那我的这些不堪真的是不堪么，我有做努力么？

有，但是不够，我需要认真的努力。付出十分的努力。

狼叔再见

发表于 2017-03-05 更新于 2020-03-17 分类于 观点 Disqus：

晚上和猫大爷一起看了『金刚狼 3：殊死一战』，据说是金刚狼系列的最后一步。剧情看下来估计往下也没法拍了，狼叔已经死了，脑洞大爷查尔斯也被弄死了，狼叔应该就这样退出历史舞台了。

电影很一般，狼叔从头到尾都瘸着一条腿，有几个场景觉得自己在看《行尸走肉》，好尴尬。打戏总觉得狼叔好可怜，谁都打不过，人到晚年也不能这样被人欺负啊。心酸死了。

感情戏也蜜汁尴尬，查尔斯大爷好容易睡个好觉，醒来就被扎心了。反派大 BOSS 是个复制版的金刚狼，然后为了剧情吧，一条新的故事线出来了一个小女孩——狼叔的女儿，所以一大半的打戏就是狼叔的女儿，狼叔，以及复制版狼叔，三个人六只钢爪『刷刷刷刷』，各种扎心。

看得好累，小狼女和狼叔的感情戏也是生拉硬拽，尴尬。就酱，不多说了。
