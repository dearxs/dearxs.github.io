Quarantined Day20

Posted on 2020-02-16 In 日记

20:22，@time

第二十天。早上起来，太阳很好，拉开窗帘，一片阳光洒进。大风降温，把雾霾什么的都吹走了。开阔，但是，确实冷。无聊的很，去 EdX 找了一个前端开发的课程学习，讲 Html&CSS ，其实之前都了解过，只是系统的学习一下，到了晚上已经到了 Module3 讲图片和表格的处理了。主要学习了：

  • HTML 和 CSS 如何组合起来工作的。
  • VS Code 安装了几个相关的插件，写起来很快。
  • HTML 的 padding 和 margin 的区别。
  • HTML 的 class 和 ID。
  • CSS 的 selector， pseudo selector？ 以及 Cascading （这个没有搞很懂）。

然后，觉得这几天情绪比较低落，所以就打开电视看了看昨天的 新闻联播 。很管用。（微笑）
接着去做饭，吃了酸辣粉，加饺子，加面条。吃到后面已经觉得不好吃了。太酸了。

然后，出门去超市采购，看到楼下新贴出来的通知，健康码可以出入小区了，不过还是需要身份证。之前发下来的出入证已经不用了。疫情似乎控制下来了。至少这里来说是这样的。

然后，就着外卖看了十几年前大学时候看的 PBS 的纪录片，China in Revolution 。还是会触动。

估计复工也就是一周之内的事情了。希望英国签证申请中心也尽快复工。

Quarantined Day19

Posted on 2020-02-15 In 日记

1 月 28 日回国，到现在 19 天的时间，基本都在家里待着。隔离。全国有无数的人像我一样，在家里。吃饭睡觉，看电视剧，刷网页。

上班是不能的。出门有了通行证，一个家庭两个人两天能出去一次。憋的头昏脑胀了。第二本书的翻译初稿已经完成了。看完了伦敦生活的第二季，Fleabag，很不错。看完了年轻的教皇，The Young Pope，很一般。看完了布莱克书店，The Black Books，好烂的英国情景喜剧啊，梗都是又烂又俗。看过了基老伴，Vicious，真实够恶毒的，还是不错的。

然后，一直鼓捣 Hammerspoon 。看到一句话说，「能自己给自己挠痒痒真爽。」深以为然。

It is extremely satisfying to scratch your own itches.

另外，土嗨英语 也一直在弄，其实是在偷懒，彪一直在每天听写，所以就索性吧听写的东西放到公众号上，然后也没什么人看。

16:55，@time

鼓捣了半天的 Hexo 主题，无果，还是 next 最好看。现在这个主题，hexo server 总是报错，不知道为什么。错误信息指向的是分类名称，所以做了一个新的分类 QDay 。

天还在下雨。说是要降温，说是晚上可能还要下雪。

21:03，@time

出去扔垃圾，在楼下站了一会儿，天上在飘雨，不是很平坦的地面上好几处水洼，送外卖的，管理进出的小棚子不少人。天空昏暗，雨淅淅沥沥，像是赛博朋克的景象。

没几分钟就回家了。做饭，鸡胸肉和刀豆，焖面。用老抽，香。鸡胸肉提前拍打腌制一下，这样不会柴。这些天几乎把自己会做的饭全部做了一遍。

吃过饭鼓捣 Anki，搜罗 Anki 的卡片包，许久。这东西还真不是个精致的玩意儿。

桌子边打字手冷，该铺床睡觉了，电褥子是我的恩人。

我看池子

Posted on 2020-01-13 In 观点

转自池子微博。很好玩的自我吐槽。来自： 原文链接 。

/

能力越低的人，越容易对自己产生过高的评价。

池子是一河南人，这我倒是没什么看法，大家对他有看法别从这上面找角度，显得狭隘，让他逮住，小机灵一抖，他占上风。但他天天冒充北京人，这就让人讨厌了，我们不清楚，你自己心里不知道自己哪儿人吗。我们北京孩子不像他那样贼眉鼠眼的，他心里门儿清，说自己是北京人就像踮脚走路，显个儿。

咱们评价人就从业务开始抓，不拐弯抹角走花边，你们也别说我没看过他作品不能评价他，你们他妈有什么资格评价我啊。第一他没作品，录几个破综艺当自己作品吗？他好意思说吗？要脸就不能说这话。第二我看过他演出，有一次在热力猫，跟朋友聊事儿碰见他们演出，当时方家胡同两边门窗还都没封死，文化氛围自由，我没站起来骂他，但他那够不上叫脱口秀，真就是瞎胡闹。之后在老书虫一次卖票的演出，我没注意有他，凑巧也看了，勉强算是脱口秀，算是摸到脱口秀的脚后跟。国内我看的不多，但是美国的单口我看的不少，目前看，如果说世界范围内脱口秀是一个巨人，中国这块是脚底板，他就是脱口秀脚后跟的死皮，简单说就是不行。他下一句就是现在大环境不好，可赶紧闭嘴，没能耐就是没能耐，20 多岁怪大环境不好，30 岁还不得天天买刀片儿。

有人能弄好啊，北京单口有一帮人弄挺好，看过一人的现场，好像叫周奇墨，是有东西，是真实力，听说他俩还认得。你眼睛睁不开，不能好好听听，好好学学吗，身边也有李诞，你是瞎还是真觉得自己怪不错的。当时说是李诞发现的他，估计是觉得他年轻，有冲劲，那是因为人家都上学去了，可不显他年轻有个性。我也承认他是有个性，有个性就往外现吗，谁还没个性了，我闺女上初中天天练腹肌，还写诗，有个性不，我看那驴就特别有个性，推磨推挺好。

他最火那节目，吐槽大会，那不叫脱口秀，也不叫喜剧，公司给他抬上去玩一个身份差，他在火的人里面最不火，他在不火的人里面又最火，拱上台说一些蠢话，捅这个挠那个，观众看了乐，台上都觉得他是一横空出世小二逼。观众发笑是因为他的行为，不是内容，这有很大差别，就像你看猴子抽烟，你觉得好笑，抽烟这件事好笑吗，不好笑，吸烟有害，得肺癌，死得快。你是觉得，猴儿也能抽烟，绝了。他自己是猴儿都没意识到，以为自己写稿小天才，以为文字语言这块儿玩的特别好。能力越低的人，越容易对自己产生过高的评价。

大家都知道，池子特喜欢抖机灵，这个词不是我提的，是网友总结的，把他的所有能力归为抖机灵，一针见血，他也没法反驳。我看过他在网上发这个发那个，他以为自己玩摇滚玩自由，以为自己多高深多巧妙，以为谁都看不出来你想什么，还不是被人民群众制裁了，虽说你爱国不爱国倒不是多大的罪，但是你活该挨骂，群众早晚治你。

其实非要公平的说，很多人都抖机灵，文人画家演戏的卖菜的，都不例外。但是网上有两个极端，就网上有，正一个反一个，有些人正着抖机灵，令人作呕，但是挑不出毛病。另一些人是反着抖机灵，漏洞百出图一乐呵。我们这是自由的时代，你说不清是谁对谁错，都该骂，抖机灵都该骂。综上，池子该骂。

我还看过他采访，人记者问他，池子老师，想对现在的年轻人说点什么，他还就真说点什么，你自己还是年轻的臭狗屎呢你还妄想教年轻人了？首先管他叫池子老师，这是娱乐圈的臭毛病，这种年轻晚辈不能直接叫他名字吗？都管他叫老师，他听的多了，敢给别人指路你信不信。池子老师，这四个字说出口真不恶心吗？在虚拟世界有点粉丝还真以为自己说话有道理了，网友们别给他这个错觉了，让这位年轻人醒一醒，对他自己也有好处。他有一段时间还说自己是当代鲁迅，我听了天灵盖疼，他还鲁迅，他是卤七里香，当代血栓差不多，鲁迅是他亲爹都得把他活活掐死。

池子粉丝最欣赏他的一点就是真实，我能理解，忽然有这么一个说大话的博主还挺新鲜的，但是他什么过硬的能力也没有啊。逮着别人有个小瑕疵就跳起来喊，说别人多假多虚，娱乐圈怎么门没关严实让他混进来也是疏忽。你玩人家的游戏得遵守人家的规则，打乒乓球规定球必须落在桌面上，你非往别人脸上打，说，裁判，我很真实。你的病情很真实。

天天这个假，那个假，蹭热度都蹭破皮了，还说自己不是蹭热度，那我们单位有个假山你怎么不去揭发检举呢，特别假的假山。有点本事，业务能力过硬，那你算个人物，最怕这种人，什么都不会，脑门上刻两个字，真实，生怕别人不知道。我觉得猪还真实呢，吃饱了睡，睡好了吃，长胖了被人吃，烤乳猪整个端上来给你看，一辈子就在这盘子里，是不是比你真实，人家也就临死了喊两声，你呢，你比猪会上网是真的。

你要是跟他讲道理，问他为什么这样，问到核心问题，我给你打包票，他会说，我就是个傻逼。他算盘打得精啊，他都说自己是傻逼了，你有什么办法，看似把自己放到大便里面，你就不敢碰他，但是他心里觉得你们是大便。要记住，他是傻逼，也是大便。

不写了，恶心他，我是看他最近消停了写两笔，要不然网友说我蹭他热度，那多恶心，网友比池子还恶心。
