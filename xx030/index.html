MacOS 上的平铺窗口管理器 Amethyst 使用

Posted on 2020-02-29 In 利器

最近在 Linux 下面鼓捣 Tiling Window Manager，Tile 就是瓷砖，Tiling 就想象成在地上铺瓷砖，一个贴一个这样。所以平铺窗口管理器就是：

把桌面上的窗口一个个地像瓷砖一样铺满桌面。

Linux 下面有好多这种 Window Manager （简称 WM），比如 i3wm，比如 awesome，比如 Sway。苹果下面就比较少了，大家也都说苹果 MacOS 的窗口管理垃圾，类似的窗口管理有 yabai ，这个有点太复杂，所以我就用了一个比较简单的 Amethyst 。

安装好，没有指南，没有设置文件，就直接启用了。然后看看设置，其实只需要记住几个快捷键就可以了。我把所有的快捷键都关了，只留下了四个：

日常使用这四个，完全够了。其他的如果需要再打开快捷键就可以。
